# Quiz Frontend

Interface utilisateur de **The Quiz of SP**.

---

# Technologies

- React
- React Router
- Vite
- SCSS

---

# Installation

npm install

---

# Variables d'environnement

Créer un fichier `.env`

VITE_API_URL=http://localhost:5050

---

# Lancer l'application

npm run dev

Application disponible sur

http://localhost:5173

---

# Fonctionnalités

- quiz multi-niveaux
- timer sur certaines questions
- mécanique quitte ou double
- leaderboard en ligne
- interface style arcade
